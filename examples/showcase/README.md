# Logan creative showcase

Golden paths you can ask Logan to run (skills must be installed under `~/.logan/skills/`).

## 1. Mouse-scrub landing

```text
/site mouse /path/to/hero.mp4 brand notes here
```

Uses **cinematic-scrub-landing** → `~/Documents/yuv-projects/landings/<slug>/`.

## 2. Scroll-scrub parallax

```text
/site parallax /path/to/clip.mp4
```

Uses **parallax-landing-page** (parasites-style one-viewport scrub).

## 3. Captioned reel

```text
/reel /path/to/talking-head.mp4 Hebrew + English captions
```

Uses **video-edit** + HyperFrames (pause for transcript review).

## 4. Learn mode

```text
/think full
explain how the scrub landing encodes frames
```

Uses **yuvai-thinking**.

Reference live craft: [yuv.ai](https://yuv.ai) · [effects.yuv.ai](https://effects.yuv.ai) · [ai-agents-skills](https://github.com/hoodini/ai-agents-skills).
