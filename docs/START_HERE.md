# Start here

## Install (one command)

**macOS / Linux**

```bash
curl -fsSL https://raw.githubusercontent.com/hoodini/logan-cli/main/scripts/install-logan.sh | bash
```

**Windows (PowerShell)**

```powershell
irm https://raw.githubusercontent.com/hoodini/logan-cli/main/scripts/install-logan.ps1 | iex
```

## Login + open

```bash
logan login
logan
```
